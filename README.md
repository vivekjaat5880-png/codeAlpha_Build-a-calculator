# CodeAlpha Calculator

A stylish web-based calculator built with **HTML, CSS, and JavaScript**.  
This project was created as part of the **CodeAlpha Internship Task**.  
It supports both **basic arithmetic** and **scientific functions**, along with **keyboard input** for smooth usage.

---

## 🚀 Features
- Basic operations: Addition, Subtraction, Multiplication, Division
- Scientific functions: sin, cos, tan, log, π, e, factorial
- Keyboard input support
- Responsive design with modern UI styling

---

## 📂 Project Structure
- `index.html` → Main HTML file
- `style.css` → Styling and layout
- `script.js` → Calculator logic and functionality
